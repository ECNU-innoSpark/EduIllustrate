# problem_103_math_g9

**题目描述：**
如图所示，已知正方形 $ABCD$ 的面积为 25，菱形 $PQCB$ 的面积为 20。求阴影部分的面积。

**选项：**
A. 11
B. 6.5
C. 7
D. 7.5

**解题思路：**
为了求出阴影部分的面积，我们可以使用割补法（减法）。阴影区域是正方形 $ABCD$ 的一部分。具体来说，它是正方形的面积减去位于正方形内部的非阴影（白色）区域的面积。通过分析正方形和菱形的性质，我们可以确定这个非阴影区域的几何形状和尺寸，并从正方形的总面积中减去它的面积。

![](scene1.png)

**步骤 1：确定边长。**

首先，我们来看正方形 $ABCD$。已知其面积为 25。
由于正方形的面积公式为 $side^2$，我们可以计算出边长：
$$BC = \sqrt{25} = 5$$

接下来，考虑菱形 $PQCB$。根据定义，菱形的四条边长度相等。因为它与正方形共用边 $BC$，所以菱形的边长也是 5。
$$PB = PQ = QC = BC = 5$$

**步骤 2：确定菱形的高。**

已知菱形 $PQCB$ 的面积为 20。菱形（作为平行四边形的一种）的面积计算公式为：
$$\text{Area} = \text{Base} \times \text{Height}$$

以 $BC$ 为底：
$$20 = 5 \times h$$
$$h = 4$$

这个高 $h$ 代表直线 $PQ$ 到直线 $BC$ 的垂直距离。在我们的坐标系中，这意味着点 $P$ 和 $Q$ 的纵坐标为 4。

![](scene2.png)

**步骤 3：确定点 P 的水平位置。**

我们需要找到点 $P$ 的确切位置，以确定非阴影区域的形状。考虑由点 $P$ 向底边 $BC$ 作垂线所形成的直角三角形。设垂足为 $F$。

在直角三角形 $\triangle PFB$ 中：
- 斜边 $PB = 5$（菱形的边长）。
- 高 $PF = 4$。

利用勾股定理（$a^2 + b^2 = c^2$）：
$$FB^2 + 4^2 = 5^2$$
$$FB^2 + 16 = 25$$
$$FB^2 = 9$$
$$FB = 3$$

因此，点 $P$ 位于点 $B$ 水平方向 3 个单位处。

**步骤 4：分析非阴影区域（梯形 $BCEP$）。**

正方形内部的非阴影区域是多边形 $BCEP$。让我们来确定它的形状：
- 边 $BC$ 是水平的，长度为 5。
- 边 $PE$ 是直线 $PQ$ 的一部分，与 $BC$ 平行。
- 因此，$BCEP$ 是一个梯形。

我们需要上底 $PE$ 的长度。
正方形的总宽度为 5（直线 $AB$ 到直线 $CD$ 的距离）。
点 $P$ 位于直线 $AB$ 右侧 3 个单位处。
因此，点 $P$ 到正方形右边缘（$CD$）的距离为：
$$PE = \text{Total Width} - \text{Horizontal Offset of P}$$
$$PE = 5 - 3 = 2$$

![](scene3.png)

**步骤 5：计算非阴影梯形的面积。**

现在我们计算梯形 $BCEP$ 的面积：
$$\text{Area}_{\text{trapezoid}} = \frac{(\text{Top Base} + \text{Bottom Base}) \times \text{Height}}{2}$$
$$\text{Area}_{\text{trapezoid}} = \frac{(2 + 5) \times 4}{2}$$
$$\text{Area}_{\text{trapezoid}} = \frac{7 \times 4}{2}$$
$$\text{Area}_{\text{trapezoid}} = 14$$

**步骤 6：计算阴影部分的面积。**

最后，我们从正方形的总面积中减去非阴影梯形的面积，从而求出阴影区域的面积。

$$\text{Area}_{\text{shaded}} = \text{Area}_{\text{square}} - \text{Area}_{\text{trapezoid}}$$
$$\text{Area}_{\text{shaded}} = 25 - 14$$
$$\text{Area}_{\text{shaded}} = 11$$

**结论：**
阴影部分的面积为 11。

**验证：**
- 正方形面积 = 25。
- 非阴影区域由一个矩形 (3x4) 缺失一个三角形 (3x4/2 = 6) 构成？不，让我们简单点看。
- 非阴影部分由一个 $2 \times 4$ 的矩形（右侧）和一个底为 $3$、高为 $4$ 的三角形组成。
- 矩形面积 = $2 \times 4 = 8$。
- 三角形面积 = $0.5 \times 3 \times 4 = 6$。
- 总非阴影面积 = $8 + 6 = 14$。
- 阴影面积 = $25 - 14 = 11$。
计算成立。

**最终答案：**
正确选项是 **A**。
