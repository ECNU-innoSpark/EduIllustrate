# problem_39_math_g6

**题目描述：**
如图所示，小旗的顶点用数对 $(5, 7)$ 表示。现将小旗绕点 $O$ 顺时针旋转 90°。此时小旗的顶点可以用数对 (   ) 表示。

**选项：**
A. $(7, 5)$
B. $(10, 3)$
C. $(9, 3)$
D. $(1, 3)$

**解题思路：**
要解决这个问题，我们首先需要通过分析网格来确定旋转中心点 $O$ 的坐标。然后，我们将顶点绕点 $O$ 进行几何旋转，以找到它的新坐标。

![](scene1.png)

**步骤 1：确定旋转中心点 $O$ 的坐标。**

题目指出旗子的顶部顶点位于 $(5, 7)$。旗杆是从该顶点向下延伸的垂直线段。观察图中的网格，我们可以通过数格子的单位来找到旗杆底部点 $O$ 的位置。

顶点的纵坐标（y坐标）为 7。从顶点向下数网格线到点 $O$：
1.  从 $y=7$ 到 $y=6$（1 个单位）
2.  从 $y=6$ 到 $y=5$（2 个单位）
3.  从 $y=5$ 到 $y=4$（3 个单位）
4.  从 $y=4$ 到 $y=3$（4 个单位）

点 $O$ 位于顶点正下方 4 个单位处。由于顶点的横坐标是 $x=5$，点 $O$ 的横坐标也是 $x=5$。因此，点 $O$ 的坐标为 $(5, 3)$。

![](scene2.png)

**步骤 2：将顶点绕点 $O$ 顺时针旋转 90°。**

现在我们进行旋转。我们将旗子绕中心点 $O(5, 3)$ 顺时针旋转 90°。

目前，从 $O$ 到顶点的连线是垂直的，向上延伸 4 个单位。
-   当前方向：向上（y 轴正方向）。
-   旋转：顺时针 90°。
-   新方向：向右（x 轴正方向）。

到中心点 $O$ 的距离必须保持不变（4 个单位）。因此，顶点的新位置将在点 $O$ **右侧** 4 个单位处。

![](scene3.png)

**步骤 3：计算新坐标。**

从中心点 $O(5, 3)$ 开始，沿水平线向右移动 4 个单位。

-   **新横坐标（x坐标）：** $5 + 4 = 9$
-   **新纵坐标（y坐标）：** 在水平移动过程中，垂直位置相对于 x 轴没有变化，所以 $y = 3$。

因此，顶点的新坐标是 $(9, 3)$。

![](scene4.png)

**结论：**

将旗子绕点 $O(5, 3)$ 顺时针旋转 90° 后，原本位于 $(5, 7)$ 的顶点移动到了 $(9, 3)$。

将此结果与给定的选项进行比较：
A. $(7, 5)$
B. $(10, 3)$
C. $(9, 3)$
D. $(1, 3)$

正确选项是 **C**。
