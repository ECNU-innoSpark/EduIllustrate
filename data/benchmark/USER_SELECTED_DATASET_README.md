# User Selected Dataset 测试集说明文档

## 📋 概述

本测试集（`user_selected.json`）是一个经过人工筛选的多学科教育题目数据集，涵盖小学、初中、高中三个学段的多个学科。数据集共包含 **118 道题目**，每道题目都包含完整的题目信息、图片、解答过程和知识点标注。

**文件路径**：`/inspire/hdd/project/ai4education/bishuzhen-CZXS24220022/edubench/TheoremExplainAgent/data/test/user_selected.json`

**数据格式**：JSON 数组格式

## 📁 数据结构说明

每道题目包含以下字段：

```json
{
  "hash_id": "题目唯一标识符",
  "img": "base64编码的图片数据",
  "question": "题目内容（可能包含LaTeX公式和图片引用）",
  "format_answer": {
    "format_solution": ["解题步骤数组"],
    "ground_truth": ["标准答案数组"]
  },
  "img_caption": "图片的文字描述",
  "difficulty": "难度级别（如：难、较难、中等、容易）",
  "type": "题目类型（如：选择题、问答题、填空题）",
  "subject": "学科代码（如：physics-g12）",
  "knowledge_point": ["知识点列表"],
  "index": "题目索引（从0开始）"
}
```

## 📈 知识点分布特征

### 数学学科特点
- **高中数学**：侧重解析几何（圆锥曲线）、立体几何、数列等
- **初中数学**：重点在几何变换、函数、相似三角形等
- **小学数学**：主要考查空间想象能力（三视图、展开图）

### 物理学科特点
- **高中物理**：集中在力学（动能定理、动量守恒）和电磁学（电磁感应、带电粒子运动）
- **初中物理**：涵盖力学基础、简单机械、电路分析等

### 化学学科特点
- **高中化学**：原电池、电解池、晶体结构、化学平衡
- **初中化学**：物质构成、化学反应基本类型

### 生物学科特点
- **高中生物**：遗传规律、分子生物学实验技术
- **初中生物**：生物发育、生理系统

### 地理学科特点
- **高中地理**：地球运动
- **初中地理**：经纬网、地球运动基础
